 📝 Blog Summariser

This is a simple Blog Summariser built for Nexium Assignment 2. It allows users to paste blog content and get a short summary.

 📌 Features

- Paste your blog/article
- Click “Summarise”
- Get a quick summary (first 2 sentences)

 🚀 Technologies Used

- HTML
- CSS
- JavaScript (Vanilla)
